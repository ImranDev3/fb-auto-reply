📖 How to Use AutoReply Pro

1 Create Your Account

Getting started with AutoReply Pro takes less than 2 minutes.

  1. Go to AutoReply Pro Sign Up Page
  2. Click on the "Sign Up" tab
  3. Enter your Full Name, Email, and Password (minimum 6 characters)
  4. Click "Create Account"
  5. You'll be automatically redirected to your Dashboard

🎉 Free Trial Included! Every new account gets the Free plan with 5 auto-reply rules. No credit card required.

2 Access Your Dashboard

After login, you'll see your main dashboard with these sections:

  • 📊 Dashboard — Overview stats + Quick Add Rule form
  • 📋 Rules — View, filter, edit, delete all your auto-reply rules
  • ⚙️ Settings — Bot settings (Away Mode, Default Reply, Greeting)
  • 👤 Profile — Your account info, Page details, Password change, Subscription

💡 Tip: Use the sidebar navigation on the left to switch between sections. On mobile, tap the hamburger menu (☰) to open the sidebar.

3 Connect Your Facebook Page

To receive and auto-reply to messages, you need to connect your Facebook Page.

3.1 — Create a Facebook Developer App

  1. Go to developers.facebook.com
  2. Click "My Apps" → "Create App"
  3. Choose "Engage with customers on Messenger from Meta"
  4. Enter app name (e.g., "My Auto Reply Bot")
  5. Complete the setup

3.2 — Get Your Page Access Token

  1. In your Facebook App, go to Messenger → Settings
  2. Under "Access Tokens", select your Facebook Page
  3. Click "Generate Token"
  4. Copy the token

3.3 — Add Token to AutoReply Pro

  1. Go to your AutoReply Pro Dashboard → Profile section
  2. Under "Page & API Details":
  3. Enter your Page Name
  4. Enter your Page ID (found in Facebook Page settings → About)
  5. Paste your Page Access Token
  6. Click "Save Page Details"

3.4 — Set Up Webhook

  1. In Facebook App → Messenger → Settings → Webhooks
  2. Click "Add Callback URL"
  3. Enter these values:
Callback URL: https://fb-auto-reply-2kst.onrender.com/webhook Verify Token: my_secret_verify_token
  1. Click "Verify and Save"
  2. Subscribe to: messages (check the box)

⚠️ Important: The Verify Token must match exactly. If you changed it in your settings, use that value instead.

4 Set Up Auto-Reply Rules

Rules are the heart of AutoReply Pro. Each rule has a keyword and a reply message.

How to Add a Rule:

  1. Go to Dashboard
  2. In the "Quick Add Rule" form:
  3. Enter a Keyword (e.g., "price", "hello", "hours")
  4. Enter the Auto Reply Message (what the bot should respond)
  5. Select Platform (Both, Messenger Only, or WhatsApp Only)
  6. Click "Add Rule"

How Matching Works:

  • Keywords are case-insensitive (Hello = hello = HELLO)
  • The bot checks if the keyword exists anywhere in the message
  • Example: Keyword "price" will match "What's your price?" or "price list please"
  • First matching rule wins (if multiple keywords match)

Example Rules:

helloHi! 👋 Thanks for reaching out. How can I help you?Both
priceOur plans start at $9.99/month. Visit our website for details!Both
hoursWe're open Mon-Fri, 9 AM - 5 PM (GMT+6). Feel free to leave a message!Both
orderTo place an order, please visit our shop: [your link]. Need help? Just ask!Messenger
thanksYou're welcome! 😊 Let us know if you need anything else.Both

💡 Pro Tip: Add common questions as keywords. Think about what your customers ask most frequently.

5 Configure Bot Settings

Go to the Settings section to configure how your bot behaves.

Available Settings:

SettingDescriptionRecommended
Auto Reply (Master Switch)Turns the entire bot ON or OFF. When OFF, no replies are sent.ON ✅
Away Mode (24/7)When ON, sends a default reply even if no keyword matches. Bot works without you being online.ON ✅
Greeting MessageSends a welcome message to first-time users.ON ✅

Custom Messages:

  • Default Reply: Sent when no keyword matches (e.g., "Thanks for your message! We'll reply soon.")
  • Away Message: Sent when you're away (e.g., "We're currently offline. We'll get back to you within 24 hours.")
  • Greeting Message: Welcome message for new users (e.g., "Welcome! 👋 How can we help you today?")

🌙 Away Mode = 24/7 Bot: With Away Mode ON, your bot ALWAYS replies — even at 3 AM. Your customers never wait for a response!

6 Connect WhatsApp (Pro Plan)

WhatsApp auto-reply is available on the Pro and Enterprise plans.

Setup Steps:

  1. In your Facebook Developer App, add the WhatsApp product
  2. Set up a WhatsApp Business phone number
  3. Get your WhatsApp Phone Number ID and Access Token
  4. In AutoReply Pro → Profile → Page & API Details:
  5. Enter WhatsApp Phone Number ID
  6. Enter WhatsApp Access Token
  7. Click "Save Page Details"

WhatsApp Webhook:

Callback URL: https://fb-auto-reply-2kst.onrender.com/whatsapp/webhook Verify Token: my_secret_verify_token Subscribe to: messages

⚠️ Note: WhatsApp Business API requires a verified business. Make sure your Meta Business account is set up properly.

7 Test Your Bot

After setup, test your bot to make sure everything works:

Testing Messenger:

  1. Open Facebook Messenger
  2. Send a message to your connected Facebook Page from a different account
  3. Type a keyword you've set up (e.g., "hello")
  4. You should receive the auto-reply within seconds! 🎉

Testing WhatsApp:

  1. Open WhatsApp
  2. Send a message to your WhatsApp Business number
  3. Type a keyword you've set up
  4. Auto-reply should come back instantly!

✅ Success! If you receive the auto-reply, your bot is working perfectly. It will now reply 24/7 automatically!

9 Troubleshooting & FAQ

❓ Bot is not replying to messages

  • Check if Auto Reply (Master Switch) is ON in Settings
  • Verify your Page Access Token is correct in Profile
  • Make sure the Webhook is properly set up in Facebook Developer Console
  • Check that you've subscribed to "messages" in webhook settings

❓ Webhook verification failed

  • Make sure the Verify Token matches exactly: my_secret_verify_token
  • Check that the Callback URL is correct and the server is running

❓ I reached my rule limit

  • Free plan allows 5 rules. Upgrade to Pro for 100 rules or Enterprise for unlimited.
  • You can delete old rules to make space for new ones.

❓ Can I use this for multiple pages?

  • Enterprise plan supports multiple pages. Free and Pro plans support 1 page each.

❓ Is my data secure?

  • Yes! All data is encrypted and isolated per user. No one else can access your rules or tokens.
  • We use JWT authentication and bcrypt password hashing.

❓ How do I cancel my subscription?

  • You can cancel anytime from your Profile page. Your account will revert to the Free plan.
